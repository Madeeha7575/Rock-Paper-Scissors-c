# Rock-Paper-Scissors-c
This project is a simple Rock-Paper-Scissors game built in C Programming.

it demonstrates the use of:

if-else statements
srand(time(0)) and rand() for random number generation
printf() and scanf() for input/output

How to Run:
1.Compile the porgram using a C compiler
2.Run the executable.
3.Follow on-screen instructions to play!
